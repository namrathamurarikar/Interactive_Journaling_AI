## 🐛 If You Get Stuck

**Error in terminal?**
1. Read the error message carefully
2. Google the error + the package name
3. If still stuck, check CURSOR_IMPLEMENTATION.md → "Troubleshooting"

**App won't start?**
- Press Ctrl+C to stop
- Run: `streamlit run app.py` again
- If still broken, check `.env` file has API key

**API key error?**
- Make sure `.env` file is in the root folder (same as `brain.py`)
- Format should be: `GEMINI_API_KEY=AIzaSyD...` (no quotes, no spaces)
- Restart terminal after saving `.env`
